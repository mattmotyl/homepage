How to cite these comparisons

Pair the visuals with qualitative context from platform transparency reports, civil society audits, and EDMO research briefs. When in doubt, link directly to the original API or dataset so readers can inspect the raw evidence.

Highlight strengths

Celebrate platforms that publish granular enforcement metrics or robust ad libraries. Positive reinforcement encourages continued investment.

Flag gaps responsibly

When a score is low, explain whether the limitation is technical, legal, or a policy choice. Suggest specific disclosures that would improve accountability.

Invite collaboration

Use the data to start conversations with regulators, academics, and civil society partners who can collectively advocate for better access.
